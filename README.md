# Heart Failure Data Analysis

## Overview
This project analyzes a heart disease dataset to identify patterns and risk factors associated with heart disease.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset
- 918 patient records
- 12 features including age, cholesterol, heart rate, and clinical indicators

## Key Insights
- Patients with heart disease showed different max heart rate patterns compared to those without
- Age appears to influence heart disease risk
- Certain clinical variables (such as chest pain type and exercise-induced angina) show strong relationships with heart disease

## Project Structure
- `heart.csv` – dataset
- `heart_failure_analysis.ipynb` – full analysis notebook
